## 5. SEO TÉCNICO — CHECKLIST

### ✅ On-Page (por implementar en cada página)

- [ ] H1 único por página con keyword primaria
- [ ] Meta title: 50–60 caracteres con keyword
- [ ] Meta description: 150–160 caracteres con CTA
- [ ] Alt text en todas las imágenes (incluir keyword)
- [ ] URLs amigables: `/aceite-oliva`, `/certificaciones`, `/productores`
- [ ] Schema Markup: `Organization`, `Product`, `BreadcrumbList`
- [ ] Open Graph tags (Facebook/Instagram preview)
- [ ] Canonical tags
- [ ] Estructura de encabezados correcta: H1 → H2 → H3

### ✅ Técnico

- [ ] Sitemap XML (`/sitemap.xml`)
- [ ] robots.txt
- [ ] Velocidad de carga < 3s (imágenes optimizadas WebP)
- [ ] Mobile-first responsive
- [ ] HTTPS (✅ ya activo en Vercel)
- [ ] Core Web Vitals: LCP < 2.5s, FID < 100ms, CLS < 0.1
- [ ] Google Search Console configurado
- [ ] Google Analytics 4 instalado

### ✅ Estructura de URLs recomendada

