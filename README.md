## 🚀 Deploy

Il sito è deployato su **Vercel** con deploy automatico dalla repository Git.

```bash
# Per testare in locale (non serve build):
# Apri index.html direttamente nel browser oppure usa un server locale:
npx serve .
```

## 🎨 Tech Stack

| Tecnologia | Uso |
|-----------|-----|
| HTML5 | Struttura semantica tutte le pagine |
| CSS3 Vanilla | Stili (index.css + inner.css) |
| JavaScript Vanilla | Lingua switcher IT/EN, form handler |
| FormSubmit.co | Gestione form contatti (no backend) |
| Google Fonts | Inter + Cormorant Garamond |
| Vercel | Hosting e deploy continuo |
